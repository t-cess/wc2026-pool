# กติการอบน็อกเอาต์ (แพ้คัดออก — รอบ 32 เป็นต้นไป)

ผล / สกอร์ / คนยิง ตัดสินที่ **90 นาที** (หมดเวลาปกติ) — ต่อเวลาและลูกโทษไม่นับในสามส่วนนี้ เพิ่มมุม **"ทีมเข้ารอบ"**

1. **ผลที่ 90 นาที** — ชนะถูก +1 / เสมอถูก +2
   คู่ที่ 90 นาทีเสมอแล้วไปต่อเวลา/ยิงโทษ นับเป็น **เสมอ** ในวง (สกอร์ตอนจบทีวีไม่ใช่ตัวตัดสิน)

2. **สกอร์เป๊ะที่ 90 นาที** +3 (ประตูช่วงต่อเวลาไม่นับ)

3. **คนยิง** +1 — คนที่ทายต้องยิงใน 90 นาที (ต่อเวลา/ลูกโทษไม่นับ) · ทาย "0-0 ไม่มีคนยิง" แล้วจบ 90 นาที 0-0 จริง = +1

4. **ทีมเข้ารอบ** +1 (ดูผลจริง รวมต่อเวลา/ลูกโทษ)
   - ทายสกอร์ให้ทีมหนึ่งชนะ → ล็อกทีมนั้นเป็น "ทีมเข้ารอบ" อัตโนมัติ ผ่านจริง = +1
   - ทายเสมอ → เลือกทีมเข้ารอบเองได้ (ช่องเสริม) เลือกถูก = +1

> ทายผล 90 นาทีพลาด ยังมีลุ้นทีมเข้ารอบเก็บแต้มตอนต่อเวลา/ยิงโทษ

## ตัวอย่าง
- ทาย "บราซิล 2-1" → 90' บราซิลชนะ 2-1 = +1 ผล, +3 สกอร์, +1 คนยิง(ถ้าถูก), +1 เข้ารอบ (ล็อกบราซิลอัตโนมัติ)
- ทาย "1-1 เลือกบราซิลเข้ารอบ" → 90' เสมอ 1-1 แล้วบราซิลชนะจุดโทษ = +2 ผล, +3 สกอร์, +1 เข้ารอบ
- ทาย "โครเอเชียชนะ 1-0" → 90' เสมอ 0-0 ต่อเวลายัน โครเอเชียเข้ารอบ = ผล/สกอร์พลาด, +1 เข้ารอบ (ล็อกโครเอเชีย)

## หมายเหตุเชิงเทคนิค (auto-grade)
- สกอร์ 90' จาก ESPN summary: `header.competitions[0].competitors[].linescores[0] + [1]` (สองครึ่งปกติ) — คู่ต่อเวลามี 4-5 ช่อง แต่บวกสองช่องแรกเสมอ = 90'
- ทีมเข้ารอบ = ผู้ชนะจริง (รวม ET/โทษ) จาก ESPN
- คนยิง 90' = goals ที่ `period.number <= 2` (ตัด ET + ลูกโทษ)
